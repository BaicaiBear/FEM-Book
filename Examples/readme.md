## Example-4-3
例4-3 考虑矩形截面悬臂梁问题， 其长$L=5.0$，高$h=0.5$，宽$b=0.1$。 梁的左端固定，右端受一力偶（$P=1$）作用。取泊松比$\nu=0$，其位移和应力的解析解为
$$ u(x)=\frac{Ph}{EI}xy,\quad v(x)=-\frac{Ph}{2EI}x^{2},\quad\sigma(x)=\frac{Ph}{I}y $$
式中$I=bh^{3}/12$为截面惯性矩。

提供了ABAQUS和NairnFEA的输入数据文件。
